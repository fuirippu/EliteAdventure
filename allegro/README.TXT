

    Allegro directory for newkind project
    =====================================

newkind uses the Allegro Game Library (www.allegro.cc), version 4.
Later versions of Allegro are NOT backwards compatible.

This directory contains a build of Allegro version 4.4.2, as well as
the source code to build Allegro version 4.4.2.

This version of Allegro can be used to build newkind.


    Contents of this directory
    --------------------------

      build_vs15/        VS2015 build of Allegro and headers
                           required to build newkind.

      allegro-4.4.2.zip  Source code distribution of Allegro.
